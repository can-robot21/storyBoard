# ImageBoard PDF 레이아웃 개선 사항

## 적용된 변경 사항

### 1. 페이지 크기 변경
- **기존**: A4 (210mm x 297mm)
- **변경**: 4A 기준 4:3 비율 (210mm x 280mm)
  - 작은축(210mm) 기준
  - 4:3 비율 = 210mm x 280mm

### 2. 레이아웃 구조

#### 상단 정보 (매 페이지 상단)
- 헤더 테이블: 2열 구성
- 시간, 씬 정보 표시
- 배경색 및 테이블 라인 적용

#### 타이틀 (상단 중앙)
- 페이지 상단 정보 아래 위치
- 중앙 정렬
- 큰 폰트(20pt) 사용

#### 본문 영역 (중간)
- 3x3 그리드 레이아웃
- 이미지 + 하단 설명
- 각 항목: 컷번호 + 이미지 + 설명

#### 주요내용 (가장 하단)
- 모든 페이지 하단에 출력
- 구분선 포함
- 최대 3줄 표시

### 3. 페이지네이션
- 페이지당 9개 항목 (3x3 그리드)
- 새 페이지 생성 시 상단 정보 및 타이틀 재출력

## StoryBoard PDF

### 유지 사항
- A4 사이즈 (210mm x 297mm)
