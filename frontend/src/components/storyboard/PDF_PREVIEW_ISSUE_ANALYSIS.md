# PDF 미리보기 출력 형태 문제 분석

## 현재 문제점

### 1. iframe 렌더링 문제
- **현재 상태**: `<iframe src={pdfBlob.url}>` 방식으로 PDF 표시
- **문제점**:
  - 브라우저 기본 PDF 뷰어에 의존
  - 높이가 `min-h-[500px]`로 고정되어 전체 내용이 보이지 않을 수 있음
  - PDF 내용이 제대로 로드되지 않을 수 있음
  - 페이지네이션이 제대로 반영되지 않을 수 있음

### 2. PDF 생성 데이터 검증 부족
- **문제점**:
  - 이미지가 Base64로 제대로 변환되었는지 확인 부족
  - 헤더 정보가 모든 페이지에 공통 출력되는지 확인 부족
  - 본문 내용이 페이지네이션에 맞게 분할되었는지 확인 부족

### 3. 레이아웃 문제
- **문제점**:
  - A4 비율이 제대로 반영되지 않을 수 있음
  - 이미지와 텍스트의 배치가 잘못되었을 수 있음
  - 헤더 테이블이 모든 페이지에 표시되지 않을 수 있음

## 개선 방안

### 1. PDFPreviewModal 개선
- **개선사항**:
  - iframe 높이를 동적으로 조정 (`100%` 또는 `calc(100vh - 200px)`)
  - PDF 로드 상태 표시 추가
  - 에러 처리 강화
  - PDF.js 라이브러리 사용 고려 (선택적)

### 2. PDF 생성 로직 검증 강화
- **개선사항**:
  - 이미지 로드 확인 로그 추가
  - 헤더 HTML 생성 확인
  - 본문 페이지네이션 확인
  - 생성된 PDF 페이지 수 확인

### 3. 레이아웃 및 스타일 개선
- **개선사항**:
  - A4 비율 강제 적용
  - 여백 최적화
  - 이미지 크기 조정
  - 텍스트 줄바꿈 처리
